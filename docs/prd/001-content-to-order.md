# PRD 001：内容到下单链路

## 背景

RedCart Copilot 不是普通搜索型商城，而是从内容种草开始，把用户带到商品、购物车、结算、支付和售后。

## 用户故事

作为消费者，我希望先从笔记里被内容种草，再顺畅完成下单和支付，这样我的购买决策会更贴近真实内容电商场景。

## MVP 范围

- 笔记流与笔记详情
- 笔记挂载商品
- 商品详情、SKU、库存与卖点
- 购物车增删改查
- 结算预览
- 幂等下单
- 模拟支付
- 退款申请

## 验收标准

- [ ] 笔记可以挂载一个或多个商品
- [ ] 商品详情返回 SKU 价格与库存
- [ ] 下单必须带幂等键
- [ ] 非法订单状态流转会被拒绝
- [ ] 接口契约记录在 `docs/api/openapi.yaml`
- [ ] 订单核心状态规则具备单元测试
