# PRD 003：AI Copilot

## 背景

这个项目不仅要展示产品里有 AI，还要展示开发流程和经营工具如何围绕 AI 建立闭环。

## 用户故事

作为商家，我希望 AI 能帮我生成商品卖点和经营复盘草案，从而更快迭代商品表达与经营动作，同时保留人工最终决策权。

## MVP 范围

- 根据商品属性、目标用户和评价生成卖点
- 根据经营漏斗数据生成复盘建议
- 记录输入、输出、任务状态和人工修正记录

## 验收标准

- [ ] AI Provider 通过稳定接口抽象
- [ ] Mock Provider 可在测试和 CI 中运行
- [ ] 提示词模板保存在 `ai-service/prompts/`
- [ ] AI 使用过程记录在 `AI_WORKFLOW.md`
