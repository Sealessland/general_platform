# Local Agent Registry

This directory is reserved for local agent handoff notes, role prompts, and generated run artifacts that are specific to this workspace.

Canonical project facts belong in `../docs/`. Files here should route agents to canonical docs and record temporary coordination state only when needed.

## Current Agents

- Default coding agent: read `../AGENTS.md`, then `../docs/index.md`.

## Handoff Format

When a handoff is needed, include:

- Objective
- Files changed
- Commands run
- Validation result
- Open blockers or residual risk
